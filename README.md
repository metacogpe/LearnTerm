# API 설계하기 

ICT 용어 학습 웹 서비스 

|목적        |   API 주소(요청 방법)  |   클라이언트는(C->S)   |  서버는(S->C) 
|:----------:|:---------------------:|:----------------------:|:------------:|
접속         | /         (GET)       |  줄 것 없음            |랜덤하게 카드 2장을 보여준다.
학습시작     |  /start    (GET)      |   줄 것 없음           | 10초 단위로 2장의 카드를 표시  
검색         |  /search   (GET)      |   검색 Key             | Key값 기준으로 2장의 카드 표시 
검색 후 학습 | /schstart (GET)       |  줄 것 없음            |10초 단위로 2장의 카드를 표시
